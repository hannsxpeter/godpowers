# MRR Tracker Stack Decision

## Decision

[DECISION] The example uses dependency-free Node.js fixtures and static HTML rendering so gate tests can run without package installation.

## Rationale

[DECISION] The bridge gate only needs artifact evidence, so the example stack avoids introducing runtime dependencies that are irrelevant to Phase 1.

## Validation

[DECISION] The stack artifact passes universal artifact lint checks and is consumed by `godpowers gate --tier=stack --project=examples/saas-mrr-tracker`.

## Flip Point

[DECISION] Revisit this stack when the example needs live Stripe OAuth, persisted user accounts, or more than 500 active subscriptions.
